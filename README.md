# Pie-Bakery-Sales-Dashboard

## About the dashboard

The dashboard represents the Sales Analysis for a Pie Bakery. It includes various elements highlighting -
* `KPIs`
* `Day-wise Sales`
* `Revenue, Cost, and Profit Trend`
* `Pie Flavour Demand`
* `Year-wise Sales`
